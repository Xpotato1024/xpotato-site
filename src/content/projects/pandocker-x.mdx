`Pandocker-X` は、Markdown から日本語 PDF を気持ちよく出すためのビルド環境です。ホスト側に LaTeX を直接入れず、Docker コンテナ内で完結するようにしています。

## できること

- `pdx new` でレポート雛形を作る
- `pdx build` で PDF を生成する
- Pandoc 引数や CSL、テンプレートを外部ファイルとして管理する

## 想定している用途

- 実験レポートや技術文書の下書き
- 参考文献や相互参照を含む長めの文章
- Windows / Linux / macOS で同じ使い方をしたいケース
