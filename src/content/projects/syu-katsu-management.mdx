就活で扱う企業ごとの情報を一か所で管理するための Web アプリです。単純な TODO ではなく、企業ごとの選考ステップ、日程、備考、ドキュメントまで持てるようにしています。

## 構成

- Backend: Go (`net/http`)
- Frontend: React + TypeScript + Vite
- Database: PostgreSQL
- Local Infra: Docker Compose

## 重視している点

- 企業ごとに選考フローを時系列で追えること
- カレンダー表示と統合予定表示を両立すること
- 認証方式を `local / none / proxy_header` で切り替えられること
- ローカル開発と本番導入の境目をなるべく単純にすること

## 今後の扱い

まだ育てている最中のプロジェクトですが、README とリリース運用まで含めて、継続して面倒を見ていく前提のアプリです。
